# 🤖 EchoBot 3D - ربات تشخیص اشیاء با دید سه‌بعدی

## ویژگی‌ها:
- شبیه‌سازی دریافت تصویر سه‌بعدی
- تشخیص اشیاء از تصویر
- ارائه اطلاعات درباره اشیاء
- قابل اجرا در ترموکس و قابل انتشار در GitHub
